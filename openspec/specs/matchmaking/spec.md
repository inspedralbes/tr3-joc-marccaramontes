## ADDED Requirements

### Requirement: Room Creation
The user SHALL be able to create a new game room with a unique identifier generated by the server.

#### Scenario: Create Room Success
- **WHEN** the user clicks "Create Room" in the Lobby
- **THEN** the server MUST register a new room and the client SHALL become the "Host" of that room

### Requirement: Join Room
The user SHALL be able to join an existing game room by providing its unique identifier.

#### Scenario: Join Room Success
- **WHEN** the user enters a valid Room ID and clicks "Join"
- **THEN** the server MUST add the client to the room and notify all participants with a "player_joined" event
