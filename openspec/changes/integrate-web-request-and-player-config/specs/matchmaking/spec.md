## MODIFIED Requirements

### Requirement: Room Creation
The user SHALL be able to create a new game room with a unique identifier generated by the server.

#### Scenario: Create Room Success via HTTP
- **WHEN** the user clicks "Create Room" in the Lobby
- **THEN** the Unity client MUST initiate a POST request to `/api/rooms/create`
- **AND** if successful, the server MUST return a new unique `roomId`

### Requirement: Join Room
The user SHALL be able to join an existing game room by providing its unique identifier.

#### Scenario: Join Room Success via HTTP
- **WHEN** the user enters a valid Room ID and clicks "Join"
- **THEN** the Unity client MUST initiate a POST request to `/api/rooms/join`
- **AND** if successful, the client MUST then establish a WebSocket connection to the server for that `roomId`
